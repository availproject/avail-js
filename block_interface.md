## Vocabulary & Structure

### Extrinsic Signature & Call

```
                MultiAddress (Enum)
------------------------------------------------
| Variant      | Type                          |
------------------------------------------------
| 0: Id        | AccountId (32 Bytes)          |
------------------------------------------------
| 1: Index     | u32 (compact)                 |
------------------------------------------------
| 2: Raw       | Vec<u8>                       |
------------------------------------------------
| 3: Address32 | [u8; 32]                      |
------------------------------------------------
| 4: Address20 | [u8; 20]                      |
------------------------------------------------

                MultiSignature (Enum)
------------------------------------------------
| Variant      | Type                          |
------------------------------------------------
| 0: Ed25519   | [u8; 64]                      |
------------------------------------------------
| 1: Sr25519   | [u8; 64]                      |
------------------------------------------------
| 2: Ecdsa     | [u8; 65]                      |
------------------------------------------------

               SignedExtra (Struct)
------------------------------------------------
| Field       | Type                           |
------------------------------------------------
| era         | Era (enum)                     |
------------------------------------------------
| nonce       | u32 (compact)                  |
------------------------------------------------
| tip         | u128 (compact)                 |
------------------------------------------------
| app id      | u32 (compact)                  |
------------------------------------------------


               Extrinsic Signature (Struct)
------------------------------------------------
| Field       | Type                           |
------------------------------------------------
| signer      | MultiAddress                   |
------------------------------------------------
| signature   | MultiSignature                 |
------------------------------------------------
| extra       | SignedExtra                    |
------------------------------------------------

               Extrinsic Call (Struct)
------------------------------------------------
| Field       | Type                           |
------------------------------------------------
| pallet Id   | u8                             |
------------------------------------------------
| variant Id  | u8                             |
------------------------------------------------
| call data   | Vec<u8> or String              |
------------------------------------------------
```

### Extrinsics

```
                                     Signed Extrinsic
+-------------------------------------------------+------------------------------------+
|               Extrinsic Signature               |           Extrinsic Call           |
+----------------+----------------+---------------+-----------+------------+-----------+
|   MultiAddress | MultiSignature | SignedExtra   | Pallet Id | Variant Id | Call Data |
+----------------+----------------+---------------+-----------+------------+-----------+



                                       Extrinsic
+-------------------------------------------------+------------------------------------+
|          [Optional] Extrinsic Signature         |           Extrinsic Call           |
+----------------+----------------+---------------+-----------+------------+-----------+
|   MultiAddress | MultiSignature | SignedExtra   | Pallet Id | Variant Id | Call Data |
+----------------+----------------+---------------+-----------+------------+-----------+



                                      Raw Extrinsic
+--------------------------------------------------------------------------------------+
|                                   Vec<u8> or String                                  |
+--------------------------------------------------------------------------------------+



                               Raw Extrinsic (partially decoded)
+-------------------------------------------------+------------------------------------+
|          [Optional] Extrinsic Signature         |           Extrinsic Call           |
+----------------+----------------+---------------+------------------------------------+
|   MultiAddress | MultiSignature | SignedExtra   |          Vec<u8> or String         |
+----------------+----------------+---------------+------------------------------------+



                                 Decoded Signed Extrinsic
+-------------------------------------------------+------------------------------------+
|               Extrinsic Signature               |           Extrinsic Call           |
+----------------+----------------+---------------+------------------------------------+
|   MultiAddress | MultiSignature | SignedExtra   |                  T                 |
+----------------+----------------+---------------+------------------------------------+



                                     Decoded Extrinsic
+-------------------------------------------------+------------------------------------+
|         [Optional] Extrinsic Signature          |           Extrinsic Call           |
+----------------+----------------+---------------+------------------------------------+
|   MultiAddress | MultiSignature | SignedExtra   |                  T                 |
+----------------+----------------+---------------+------------------------------------+
```

### Block Extrinsics

```
               Metadata (Struct)
------------------------------------------------
| Field       | Type                           |
------------------------------------------------
| tx hash     | H256                           |
------------------------------------------------
| tx index    | u32                            |
------------------------------------------------
| pallet id   | u8                             |
------------------------------------------------
| variant id  | u8                             |
------------------------------------------------
| block id    | u32 or String or H256          |
------------------------------------------------



                                                                 Block Signed Extrinsic
+--------------------------------------------------------------------------------------+--------------------------------------------------------+
|                               Decoded Signed Extrinsic                               |                    Extra Information                   |
+-------------------------------------------------+------------------------------------+--------------------------------------------------------+
|               Extrinsic Signature               |           Extrinsic Call           |                        Metadata                        |
+----------------+----------------+---------------+------------------------------------+---------+----------+-----------+------------+----------+
|   MultiAddress | MultiSignature | SignedExtra   |                  T                 | Tx Hash | Tx Index | Pallet Id | Variant Id | Block Id |
+----------------+----------------+---------------+------------------------------------+---------+----------+-----------+------------+----------+

         Block Signed Extrinsic Methods
+-------------------------------------------------+
| + appId(&self) -> u32                           |
| + nonce(&self) -> u32                           |
| + tip(&self) -> u128                            |
| + ss58Address(&self) -> String (nullable)       |
+-------------------------------------------------+



                                                                      Block Extrinsic
+--------------------------------------------------------------------------------------+--------------------------------------------------------+
|                               Decoded Signed Extrinsic                               |                    Extra Information                   |
+-------------------------------------------------+------------------------------------+--------------------------------------------------------+
|         [Optional] Extrinsic Signature          |           Extrinsic Call           |                        Metadata                        |
+----------------+----------------+---------------+------------------------------------+---------+----------+-----------+------------+----------+
|   MultiAddress | MultiSignature | SignedExtra   |                  T                 | Tx Hash | Tx Index | Pallet Id | Variant Id | Block Id |
+----------------+----------------+---------------+------------------------------------+---------+----------+-----------+------------+----------+

         Block Signed Extrinsic Methods
+-------------------------------------------------+
| + appId(&self) -> u32 (nullable)                |
| + nonce(&self) -> u32 (nullable)                |
| + tip(&self) -> u128 (nullable)                 |
| + ss58Address(&self) -> String (nullable)       |
+-------------------------------------------------+



                                                                                Block Raw Extrinsic
+--------------------------------------------------------------------------------------+--------------------------------------------------------------------------------------------------------------+
|                                                                                      |                                              Extra Information                                               |
|                  [Optional] Raw Extrinsic or Raw Extrinsic Call                      +--------------------------------------------------------+-----------------------------------------------------+
|                                                                                      |                        Metadata                        |               [Optional] Signer Payload             |
+--------------------------------------------------------------------------------------+---------+----------+-----------+------------+----------+--------+-------+------------------------+-----------+
|                                   String                                             | Tx Hash | Tx Index | Pallet Id | Variant Id | Block Id | App Id | Nonce | [Optional] SS58Address | Mortality |
+--------------------------------------------------------------------------------------+---------+----------+-----------+------------+----------+--------+-------+------------------------+-----------+

         Block Raw Extrinsic Methods
+-------------------------------------------------+
| + appId(&self) -> u32 (nullable)                |
| + nonce(&self) -> u32 (nullable)                |
| + ss58Address(&self) -> String (nullable)       |
+-------------------------------------------------+
```
